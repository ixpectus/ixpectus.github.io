Заметки с курсов хекслета по javascript

Инверсия зависимостей Зависимости передаются на вход в функцию или объект, в котором они необходимы. Помимо всего прочего это облегчит тестирование. Может быть так, что с точки зрения бизнес логики зависимость передавать и не надо, к примеру в может быть обращение к системной функции определения текущего времени. Но это может существенно затруднить написание тестов и можно передавать текущее время на вход функции.
